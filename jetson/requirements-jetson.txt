#       pip install --no-cache-dir torch torchvision --index-url <jetson-wheel-index>
#     설치 후 확인:  python3 -c "import torch; print(torch.__version__, torch.cuda.is_available())"
#
#   [ultralytics]
#     위 torch 설치 후, torch를 다시 끌어오지 않도록 의존성 무시로 설치:
#       pip install --no-deps ultralytics
#     (ultralytics가 요구하는 나머지 순수 파이썬 의존성은 개별 설치)
